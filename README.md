
# Kumojin Event Manager

Project made for Kumojin interview process.

Stack used:



## Stack Used

- FastApi
- SQLAlchemy - Alembic
- SQLite
- React
- Tailwind CSS
- Axios


## API Reference

#### Get all Events

```http
  GET /api/event
```

#### Post an Event

```http
  POST /api/event
```

#### Delete an Event

```http
  DELETE /api/event/{event_id}
```
